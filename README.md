An automated Python-based tool designed for electrical engineers in Egypt to calculate building loads and connection costs according to the *2020 Electricity Distribution Code*. This project bridges the gap between traditional engineering calculations and modern automation.

## üöÄ Key Features
- *Automatic Load Calculation:* Estimates design loads based on area, number of floors, and building type.
- *District-Based Logic:* Adjusts calculations according to geographical locations (Village vs. Urban) and neighborhood class (Standard, Middle, Posh).
- *Auxiliary Loads:* Seamlessly integrates power requirements for Elevators and Water Pumps.
- *Cost Estimation:* Automatically differentiates between *Standard (Flat rate)* and *Actual (Technical details)* estimation methods.
- *Transformer Room Checker:* Provides a decision-making logic on whether a dedicated transformer room is legally required.
- *Excel Integration:* Generates a professional, color-coded Excel report with all formulas active.
